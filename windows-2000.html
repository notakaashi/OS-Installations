Windows 2000 Installation Guide

Watch the Installation Process

Installation Steps

  1. Boot your computer from the Windows 2000 CD-ROM
  2. Press any key to boot from CD when prompted
  3. Wait for setup to load files and start the text-based portion
  4. Press Enter to set up Windows 2000
  5. Read and accept the license agreement by pressing F8
  6. Select an unpartitioned space or create a new partition
  7. Format the partition using NTFS (recommended) or FAT32
  8. Wait for setup to copy files to the hard drive
  9. System will restart and begin GUI-based setup
  10. Configure regional and language settings
  11. Enter your name and organization information
  12. Type your product key when prompted
  13. Set your computer name and administrator password
  14. Configure date, time, and time zone settings
  15. Wait for network setup and final configuration
  16. Your Windows 2000 installation is complete
