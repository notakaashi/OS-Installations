Windows Server 2012 Installation Guide

Watch the Installation Process

Installation Steps

  1. Boot your server from the Windows Server 2012 DVD or USB
  2. Press any key to boot from installation media
  3. Wait for setup to load files and start
  4. Select your language, time, and keyboard preferences
  5. Click "Next" and then "Install Now"
  6. Accept the license agreement by checking the box
  7. Choose "Custom (advanced)" installation type
  8. Select the installation option: Server with GUI or Server Core
  9. Select the partition where you want to install Server 2012
  10. Click "Next" to begin installation (format if needed)
  11. Wait for Windows to copy files and install features
  12. System will restart during installation process
  13. Change the administrator password when prompted
  14. Configure initial server settings
  15. Set your time zone and date/time settings
  16. Configure network settings and IP address
  17. Launch Server Manager for role administration
  18. Install required server roles and features
  19. Your Windows Server 2012 installation is complete
