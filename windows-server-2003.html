Windows Server 2003 Installation Guide

Watch the Installation Process

Installation Steps

  1. Boot your server from the Windows Server 2003 CD-ROM
  2. Press any key to boot from CD when prompted
  3. Wait for setup to load files and start text-based setup
  4. Press Enter to set up Windows Server 2003
  5. Read and accept the license agreement by pressing F8
  6. Select an unpartitioned space or create a new partition
  7. Format the partition using NTFS (required for server)
  8. Wait for setup to copy files to the hard drive
  9. System will restart and begin GUI-based setup
  10. Configure regional and language settings
  11. Enter your name and organization information
  12. Type your product key when prompted
  13. Choose your licensing mode (per server or per seat)
  14. Set your computer name and administrator password
  15. Configure date, time, and time zone settings
  16. Select typical or custom network settings
  17. Choose to join a workgroup or domain
  18. Wait for final configuration and setup completion
  19. Your Windows Server 2003 installation is complete
