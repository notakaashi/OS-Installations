About Me

Learn about the person behind these comprehensive OS installation guides

Hello! I'm Marcus Inigo S. Aristain, a passionate BSIT student from Section S-2A at the Technological University of the Philippines. I created this website as part of my journey to share knowledge about Windows operating systems.

As a student of Information Technology, I'm fascinated by how different operating systems work and evolve over time. This website showcases my understanding of Windows OS installations from the early days of Windows 95 to the modern Windows 11.

Through this project, I've gained hands-on experience in web development, content creation, and technical documentation. Each installation guide represents my research and understanding of how these systems work.

I'm currently studying at the Technological University of the Philippines, where I'm developing my skills in various areas of information technology, including web development, system administration, and technical support.

This website serves as both a learning resource for others and a demonstration of my technical abilities. I hope you find these guides helpful for your own OS installation journey.

Thank you for visiting my project. Feel free to explore the different installation guides and learn more about the evolution of Windows operating systems.
