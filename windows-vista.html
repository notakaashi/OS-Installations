Windows Vista Installation Guide

Watch the Installation Process

Installation Steps

  1. Boot your computer from the Windows Vista DVD
  2. Press any key to boot from DVD when prompted
  3. Wait for setup to load files and start
  4. Select your language, time, and keyboard preferences
  5. Click "Install Now" to begin installation
  6. Accept the license agreement by checking the box
  7. Choose "Custom (advanced)" installation type
  8. Select the partition where you want to install Vista
  9. Click "Next" to begin installation (format if needed)
  10. Wait for Windows to copy files and install features
  11. System will restart several times during installation
  12. Configure your username and password
  13. Choose your computer name and desktop background
  14. Set your time zone and date/time settings
  15. Configure Windows Update settings
  16. Complete the network and security configuration
  17. Your Windows Vista installation is now complete
