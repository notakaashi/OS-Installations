Windows 95 Installation Guide

Watch the Installation Process

Installation Steps

  1. Start your computer and boot from the Windows 95 installation disk
  2. Press F8 when you see "Starting Windows 95" to access the startup menu
  3. Select "Command Prompt Only" from the menu
  4. Navigate to your CD-ROM drive using the appropriate drive letter
  5. Run SETUP.EXE to begin the installation process
  6. Follow the on-screen instructions to accept the license agreement
  7. Choose the installation directory (default is C:\WINDOWS)
  8. Select your installation type: Typical, Portable, Compact, or Custom
  9. Enter your user information and product key when prompted
  10. Wait for files to copy to your hard drive
  11. Configure your hardware settings when the system restarts
  12. Complete the final setup steps including date/time and printer setup
  13. Your Windows 95 installation is now complete
