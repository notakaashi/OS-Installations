Windows 7 Installation Guide

Watch the Installation Process

Installation Steps

  1. Boot your computer from the Windows 7 DVD
  2. Press any key to boot from DVD when prompted
  3. Wait for setup to load files and start
  4. Select your language, time, and keyboard preferences
  5. Click "Next" and then "Install Now"
  6. Accept the license agreement by checking the box
  7. Choose "Custom (advanced)" installation type
  8. Select the partition where you want to install Windows 7
  9. Click "Next" to begin installation (format if needed)
  10. Wait for Windows to copy files and install features
  11. System will restart during installation process
  12. Configure your username and computer name
  13. Set your password and password hint
  14. Enter your product key when prompted
  15. Choose your Windows Update settings
  16. Set your time zone and date/time
  17. Select your network location (Home, Work, or Public)
  18. Your Windows 7 installation is now complete
