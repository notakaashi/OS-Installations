Windows 98 Installation Guide

Watch the Installation Process

Installation Steps

  1. Boot your computer from the Windows 98 startup disk
  2. Enable CD-ROM support from the startup menu
  3. Navigate to your CD-ROM drive (usually D: or E:)
  4. Type SETUP and press Enter to start installation
  5. Run the system scan to check for problems
  6. Accept the license agreement to continue
  7. Enter your product key when prompted
  8. Select the installation directory (default is C:\WINDOWS)
  9. Choose your setup type: Typical, Custom, or Compact
  10. Select the components you want to install
  11. Configure your computer identity and workgroup
  12. Wait for files to copy and system to restart
  13. Complete the hardware detection process
  14. Set your time zone and date/time settings
  15. Your Windows 98 installation is now ready to use
