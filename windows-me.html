Windows ME Installation Guide

Watch the Installation Process

Installation Steps

  1. Start your computer and boot from the Windows ME CD-ROM
  2. Press any key when prompted to boot from CD
  3. Choose "Start Windows ME Setup" from the menu
  4. Run the system check to verify hardware compatibility
  5. Accept the license agreement to proceed
  6. Enter your product key when prompted
  7. Select the installation directory (default is C:\WINDOWS)
  8. Choose your installation type: Typical, Custom, or Compact
  9. Select the components you want to install
  10. Enter your computer name and workgroup information
  11. Wait for setup to copy files to your hard drive
  12. System will restart and continue setup
  13. Complete the hardware detection and configuration
  14. Set your time zone and date/time preferences
  15. Configure your internet connection if needed
  16. Your Windows ME installation is now complete
